# ScreenTranslator 🌍

A lightweight, practical desktop tool built with C# and WPF that allows you to instantly capture any part of your screen, extract the text using OCR, and translate it.

## ✨ Features
- **Smart OCR:** Uses Tesseract Engine (LstmOnly) to accurately read texts, even on dark themes or complex game UI.
- **Multiple Languages:** Supports translation between English, Turkish, Russian, Spanish, French, Italian, and German.
- **API Choice:** Choose between Google Translate (Fast) and Yandex Translate (Better for game context).
- **Global Shortcuts:** Activate the capture tool anywhere using customizable hotkeys (e.g., Ctrl+Alt+X).
- **Background Mode:** Runs quietly in the system tray.

## 🛠️ Tech Stack
- C# / WPF (.NET)
- Tesseract OCR
- GTranslate (Google & Yandex API)

## 🚀 How to Use
1. Go to the **Releases** section on the right side of this page.
2. Download the latest `ScreenTranslator_v1.0.zip` file.
3. Extract the folder and run `ScreenTranslator.exe`.
4. Right-click the tray icon to set your languages and hotkey!
